## Commands

### 1- Package used to create this app:

    - npx create-react-app my-app --template pwa-typescript

### 2- Install React Router v6

    - npm install react-router@next react-router-dom@next

### 3- Install React Sass

    - npm install sass

### 4- Install axios
    - npm install axios
